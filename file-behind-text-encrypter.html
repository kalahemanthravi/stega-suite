Usage Guide

What is this?

This tool allows you to hide a secret text message inside an image file (LSB method) or attach any file/text to the end of another file (Append method).

How to use (Pixel Stealth):

  1. Select the "Pixel Stealth (LSB)" tab.
  2. Upload an image and type your secret message.
  3. Set a password and click Encrypt & Hide.
  4. Save the resulting PNG image. It contains your hidden message.
