How to Use

What is this?

This tool transforms readable text into a secure, scrambled code using military-grade AES-256 encryption. It ensures that only someone with the correct password can read your message. The process happens entirely in your browser; no data is ever sent to a server.

Steps:
  • Encrypt: Type your message, enter a password, and click 'Encrypt'. Copy the resulting code.
  • Decrypt: Paste the code into the input box, switch to 'Decrypt' mode, enter the password, and click 'Decrypt'.
