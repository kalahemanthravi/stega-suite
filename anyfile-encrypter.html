What is this tool?

This is a general-purpose file locker. It takes *any* computer file (documents, videos, zips, etc.) and locks it inside a secure digital vault (encrypted file).

To Lock a File:
  1. Upload your file.
  2. Set a password.
  3. Download the protected .enc file.
To Unlock:
  1. Switch to "Decrypt".
  2. Upload the .enc file.
  3. Enter the original password to restore your file.
