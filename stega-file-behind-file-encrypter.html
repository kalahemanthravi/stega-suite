User Guide

What is this tool?

This is a File Binding Tool. It allows you to take a "Secret File" (any format) and attach it to a "Carrier File" (Image or Video).

The resulting file will look and play exactly like the original Carrier File, but it contains your encrypted secret file hidden inside it. This functionality is often called "Append Steganography".

How to use

  1. Encrypt: Upload a Carrier (e.g., a photo) and a Secret File (e.g., a PDF). Set a password and download the result.
  2. Decrypt: Go to the "Extract" tab. Upload the secured file you created previously. Enter the password to retrieve your secret file.

Pro Tip: Use this for hiding large archives or files that wouldn't fit in standard pixel-based steganography.
