How to Use Time Lock

This tool creates a "digital time capsule". You can encrypt a file so that it is mathematically impossible to open until a specific date and time in the future, even with the password.

  • Encrypt: Upload a file, set a password, and choose an "Unlock Date". The file will be locked until that moment.
  • Decrypt: Upload the locked file (.chronos) and enter the password. If the unlock time hasn't passed, the system will prevent access.
