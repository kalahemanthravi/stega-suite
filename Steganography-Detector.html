About this Tool

This detector checks files for signatures left by the "Standard" Stega Suite tools (like File-Behind-File Append mode). It is designed to help you verify if a file might contain hidden data created by this suite.

How to use: Simply drop a file into the upload zone and click "Analyze". The tool will scan the file structure and report if any hidden data partitions are found.
