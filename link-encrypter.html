About & Instructions

What is it? This tool creates a secure "middleman" link. When someone clicks your secured link, they are asked for a password. Only after entering the correct password are they redirected to the actual destination URL.

How to use:

  1. Enter the target URL (e.g., a secret document link).
  2. Set a password.
  3. Click Encrypt Link.
  4. Share the generated link. The recipient will need the password to access the destination.
